# 体育文创设计征集平台

一个面向大学生的体育文创设计征集平台首页原型，适合用于赛事官网、校内征集活动、文创设计季或课程项目展示。

当前版本包含这些内容：

- 强视觉首屏与明确投稿入口
- 四个征集赛道的动态切换展示
- 赛程时间线、奖项设置、评审维度
- 杂志式优秀作品展示区
- 面向学生投稿场景的流程与材料说明

## 启动

```bash
npm start
```

打开浏览器访问：

```text
http://localhost:3000
```

## 结构

- `server.js`: 轻量静态文件服务
- `public/index.html`: 平台首页结构
- `public/styles.css`: 整体视觉与响应式样式
- `public/app.js`: 赛道切换、倒计时和滚动显现交互
